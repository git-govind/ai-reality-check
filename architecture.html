Evaluation Engine
🔎 Factual Checker factual_checker.py
extract_key_claims() → ≤8 factual sentences
Per claim — DuckDB first:
  keyword → LIKE query → word_overlap ≥0.40 → supported/contradicted
Fallback — Wikipedia:
  search API → extract → overlap ≥0.25
Scoring: DB=1.00 · Wiki=0.90 · unverified=0.50
contradiction penalty: −15pts each
⚖ Consistency Checker consistency_checker.py
Heuristic: 6 contradiction regex patterns
  base 100 − (hits × 10), floor 50
LLM critique: bullet filter with 3 guards
  ① bullet marker ② problem keyword
  ③ no negation before keyword & no "X: None"
Final = 0.3×heuristic + 0.7×llm_score
🛡 Bias & Safety Checker bias_safety_checker.py
6 bias rules → −20pts each (demographic generalisations, slurs…)
6 safety rules → −25pts each (weapons, self-harm, CSAM…)
LLM scan → 0.4×rule + 0.6×llm blend (bias only)
Safety score: rule-based only, never blended
✏ Clarity & Completeness clarity_scorer.py
Completeness: keyword overlap×130, capped 100
  flags: <30 words · >600 words · <30% topic match
Clarity: start 100
  avg sentence >35w → −10
  vague qualifiers → −(count×4) max−20
  filler phrases → −(count×5) max−15
Score = 0.6×completeness + 0.4×clarity

Pipeline Steps
📋 Step 1 — Metadata metadata_checker.py
EXIF via Pillow getexif()
Scoring: start 100
−30 no EXIF
−40 editing software
−20 no camera model
−10 timestamp anomaly
+20 valid camera pipeline
→ MetadataResult.score [0,1]
🔎 Step 2 — Pixel Forensics pixel_forensics.py
ELA: re-compress at q=75, pixel diff
Noise: Gaussian residual · std · CV
FFT: spectral peak ratio
JPEG quant: table suspicion check
Formula:
100−(ela×0.40+noise×0.30+fft×0.30)
→ PixelForensicsResult.score [0,1]
🤖 Step 3 — AI Artifact ai_artifact_classifier.py
6 heuristics (always runs):
texture smoothness · colour entropy
edge sharpness · HF/LF ratio
saturation skew · bilateral symmetry
FreqNet-inspired (always runs):
spectral roll-off · mid-freq anomaly
spectral entropy · peak prominence
CLIP probe (optional, SKIP_CLIP=1 to skip)
method: heuristic+freqnet[+model]
→ AIArtifactResult.ai_prob [0,1]
📝 Step 4 — Consistency image_text_consistency.py
CLIP path (preferred):
start 100 · penalty scoring:
−40 object mismatch
−30 impossible geometry
−20 lighting/physics mismatch
−10 caption hallucination (×3 max)
Keyword fallback (no CLIP):
dominant colour matching
Skipped if no caption supplied → ConsistencyResult.score [0,1]
🌐 Step 5 — Reverse Search reverse_image_search.py
Priority order:
1. Google Cloud Vision API
2. Bing Visual Search API
3. SerpApi (Google Lens)
pHash refinement on source DLs
Similarity tiers:
full match → 1.00
partial match → 0.80
pages only → 0.50
Skipped if no API key configured → ReverseSearchResult.similarity [0,1]
MetadataResult · PixelForensicsResult · AIArtifactResult · ConsistencyResult · ReverseSearchResult
