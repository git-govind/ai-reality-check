🔎 Factual Checker factual_checker.py
extract_key_claims() → ≤8 factual sentences
Per claim — DuckDB first:
  keyword → LIKE query → word_overlap ≥0.40 → supported/contradicted
Fallback — Wikipedia:
  search API → extract → overlap ≥0.25
Scoring: DB=1.00 · Wiki=0.90 · unverified=0.50
contradiction penalty: −15pts each
utils.text_utils.word_overlap
⚖ Consistency Checker consistency_checker.py
Heuristic: 6 contradiction regex patterns
  base 100 − (hits × 10), floor 50
LLM critique: bullet filter with 3 guards
  ① bullet marker ② problem keyword
  ③ no negation before keyword & no "X: None"
Final = 0.3×heuristic + 0.7×llm_score
utils.text_utils.extract_issue_bullets · parse_llm_score models/llm_registry.py
🛡 Bias & Safety Checker bias_safety_checker.py
6 bias rules → −20pts each (demographic generalisations, slurs…)
6 safety rules → −25pts each (weapons, self-harm, CSAM…)
LLM scan → 0.4×rule + 0.6×llm blend (bias only)
Safety score: rule-based only, never blended
utils.text_utils.extract_issue_bullets · parse_llm_score models/llm_registry.py
✏ Clarity & Completeness clarity_scorer.py
Completeness: keyword overlap×130, capped 100
  flags: <30 words · >600 words · <30% topic match
Clarity: start 100
  avg sentence >35w → −10
  vague qualifiers → −(count×4) max−20
  filler phrases → −(count×5) max−15
Score = 0.6×completeness + 0.4×clarity

Pipeline Steps
📋 Step 1 — Metadata metadata_checker.py
EXIF via Pillow getexif()
XMP / ICC / JFIF parsing
AI software tag detection
Scoring: start 100
−30 no EXIF on JPEG
−40 AI generator software
−20 no camera model
−15 power-of-two AI dims
−10 timestamp anomaly
+20 valid camera pipeline
editing_penalty: [0, 100]
config/thresholds.yaml
→ MetadataResult.score [0,1]
🔎 Step 2 — Pixel Forensics pixel_forensics.py
ELA: re-compress at q=75, pixel diff p95
JPEG Ghost: 5 quality levels [50–90]
  patch native-quality entropy (0→100)
  spatial CV at dominant quality
Noise (patch): Gaussian residual · std · CV
Noise (16-block): 4×4 grid CV
  real camera: low CV · AI: high CV
FFT: spectral peak ratio (log-scale)
JPEG quant: DQT table suspicion
Pixel formula:
100−(ela×0.30+ghost×0.20
  +noise×0.25+fft×0.25)
noise = 60%×patch + 40%×16-block
FFT in pixel_score only — excluded from ai_likelihood
(JPEG 8×8 block-DCT raises FFT in authentic photos too)
utils.image_utils.jpeg_recompress · coeff_of_variation
config/thresholds.yaml · weights.yaml
→ PixelForensicsResult.score [0,1]
  + confidence_band
🤖 Step 3 — AI Artifact ai_artifact_classifier.py
① HuggingFace classifier (primary):
  umm-maybe/AI-image-detector
  ViT fine-tuned real-vs-AI
  lazy-loaded, process-cached
  SKIP_HF_MODEL=1 to force heuristic
② FreqNet-inspired (always):
  spectral roll-off · mid-freq anomaly
  spectral entropy · peak prominence
③ 6 pixel heuristics (always):
  texture smoothness · colour entropy
  edge sharpness · HF/LF ratio
  saturation skew · bilateral symmetry
Combine (model available):
  ai_prob = 0.80×model + 0.20×blend
Combine (heuristic only):
  ai_prob = 0.70×heuristic + 0.30×freqnet
OOD detection → ood_warning badge
models/image_model_registry.py
→ AIArtifactResult.ai_prob [0,1]
  + confidence_band · method · ood_warning
📝 Step 4 — Consistency image_text_consistency.py
CLIP path (preferred):
clip-ViT-B-32 cosine similarity
mapped from [0.15–0.35] → [0,1]
Sub-phrase re-ranking for issues
Keyword fallback (no CLIP):
dominant colour + noun matching
models/embeddings_registry.py
Skipped if no caption supplied
→ ConsistencyResult.score [0,1]
🌐 Step 5 — Reverse Search reverse_image_search.py
Priority order:
1. Google Cloud Vision API
2. Bing Visual Search API
3. SerpApi (Google Lens)
pHash refinement on source DLs
Similarity tiers:
≥0.90 → 1.00 (exact match)
≥0.70 → 0.80 (high sim)
≥0.30 → 0.50 (partial)
<0.30 → 0.00 (no match)
Skipped if no API key configured
→ ReverseSearchResult

⚡ Image Scoring Engine image_scoring.py
Image-type detection (flat_ratio · unique_ratio · noise_std):
  photo · illustration · screenshot
Adaptive weights per type (photo defaults):
Metadata 20% Pixel 25% AI Artifact 35% Consistency 10% Reverse 10%
Skipped steps → normalize_weights() redistributes proportionally
authenticity_score = weighted_sum × 100
ai_likelihood (ML-gated pixel boost):
  gate = clip((ai_prob−0.10)/0.20, 0, 1)
  ai_prob* = ai_prob + gate×noise_block_sig×0.50×(1−ai_prob)
  signal: noise_block_consistency CV → [0,1] · FFT excluded
  gate=0 when ML confident (ai_prob<0.10) → no pixel boost
editing_likelihood = 0.6×ela_badness + 0.4×metadata_edit_penalty
Confidence band propagation: Σ(w_i × band_i) × 100
Top 3 signals ranked by strength · ai threshold ≥ 0.25
Grade: A ≥80 · B ≥65 · C ≥50 · D ≥35 · F <35
